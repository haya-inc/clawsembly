Clawsembly remote Gateway cockpit

Connect your OpenClaw: this page opens the generated, version-locked Clawsembly client against a Gateway you operate. Nothing runs in this browser beyond the bounded chat client; the connection is interoperability, not browser-local execution, and it carries no evidence claim.

Gateway connection

The Gateway admits this page only when its gateway.controlUi.allowedOrigins allowlist contains this page's exact origin () — the Gateway's own connect error names that setting. The client refuses any Gateway that is not running the verified artifact version ().
